\subsection{Detailed Description}
This class contains static variables only, which hold the configuration parameters for all other classes used by the segway project. 

In \doxyref{Configuration.\+h}{p.}{_configuration_8h_source} the variables and structs are declared. In \doxyref{Configuration.\+cpp}{p.}{_configuration_8cpp_source} the variables are defined and initialized with zero. In \doxyref{init()}{p.}{class_configuration_a83a5078966294216bb61a4dae6d42e0e} the variables are set to the configuration values.

This behavior allows calculations to be made within \doxyref{init()}{p.}{class_configuration_a83a5078966294216bb61a4dae6d42e0e}.

\subsection{Detailed Description}
Contains main segway functionality as there are initialization and the controlling algorithm. 

This class creates and initializes helper objects of sensor, motor and communication classes. Then, it uses these objects to execute the segway controlling algorithm. Debug data is provided via bluetooth using the \doxyref{U\+A\+R\+T}{p.}{class_u_a_r_t} class.
